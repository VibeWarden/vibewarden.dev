Developer tools

Open-source utilities from the VibeWarden ecosystem. Each is a standalone tool — use them with or without VibeWarden.

httptape

Record and replay HTTP traffic. Capture requests and responses as tape files for debugging, testing, and building mocks. Lightweight CLI with zero dependencies.

Documentation GitHub

snitchproxy

Transparent HTTP proxy that logs every outbound request your app makes. Catch unexpected API calls, data exfiltration, and SSRF attempts during development.

Documentation GitHub
