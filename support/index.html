Security reporting

Responsible disclosure keeps everyone safe.

Report a vulnerability

If you discover a security vulnerability in VibeWarden, please report it responsibly. Send details to security@vibewarden.dev. We will acknowledge your report within 48 hours and work with you to resolve the issue.

Please do not open public GitHub issues for security vulnerabilities. Use the email above for responsible disclosure.
